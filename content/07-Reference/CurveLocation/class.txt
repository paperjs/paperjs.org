<div class="reference-class">


<p>CurveLocation objects describe a location on <a href="/reference/curve"><tt>Curve</tt></a>
objects, as defined by the curve <a href="/reference/curvelocation#parameter" onclick="return toggleMember('parameter', true);"><tt>parameter</tt></a>, a value between
<tt>0</tt> (beginning of the curve) and <tt>1</tt> (end of the curve). If
the curve is part of a <a href="/reference/path"><tt>Path</tt></a> item, its <a href="/reference/curvelocation#index" onclick="return toggleMember('index', true);"><tt>index</tt></a> inside the
<a href="/reference/path#curves"><tt>path.curves</tt></a> array is also provided.</p>
<p>The class is in use in many places, such as
{@link Path#getLocationAt(offset, isParameter)},
<a href="/reference/path#getlocationof-point"><tt>path.getLocationOf(point)</tt></a>,
{@link Path#getNearestLocation(point),
<a href="/reference/pathitem#getintersections-path"><tt>pathItem.getIntersections(path)</tt></a>,
etc.</p>

</div>
